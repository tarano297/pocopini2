# راهنمای تنظیم دکمه واتساپ

## مشکل فعلی
دکمه واتساپ در حال حاضر با شماره نمونه `989123456789` تنظیم شده که یک شماره واقعی نیست.

## راه حل

### مرحله 1: پیدا کردن شماره واتساپ
شماره واتساپ خود را که میخواهید مشتریان با آن تماس بگیرند، آماده کنید.

### مرحله 2: فرمت صحیح شماره
شماره باید به این فرمت باشد:
- **بدون** علامت `+`
- **بدون** صفر اول
- با کد کشور

**مثال‌ها:**
- شماره ایرانی: `09123456789` → `989123456789`
- شماره ترکیه: `05551234567` → `905551234567`
- شماره آمریکا: `(555) 123-4567` → `15551234567`

### مرحله 3: تغییر شماره در کد

فایل را باز کنید:
```
frontend/src/components/WhatsAppButton.jsx
```

خط 8 را پیدا کنید:
```javascript
const whatsappNumber = '989123456789'; // ⚠️ این شماره نمونه است
```

شماره خود را جایگزین کنید:
```javascript
const whatsappNumber = '989121234567'; // شماره واقعی شما
```

### مرحله 4: تست کردن

1. فایل را ذخیره کنید
2. سایت را رفرش کنید
3. روی دکمه واتساپ کلیک کنید
4. باید واتساپ باز شود و چت با شماره شما آماده باشد

## نکات مهم

⚠️ **شماره باید روی واتساپ فعال باشد**
- شماره باید روی یک دستگاه با واتساپ نصب شده فعال باشد
- اگر شماره روی واتساپ نیست، لینک کار نمیکند

⚠️ **تست در موبایل**
- در موبایل، واتساپ اپلیکیشن باز میشود
- در دسکتاپ، واتساپ وب یا دسکتاپ باز میشود

## تغییر متن پیام پیش‌فرض

اگر میخواهید متن پیام پیش‌فرض را تغییر دهید، خط 9 را ویرایش کنید:

```javascript
const message = 'سلام، می‌خواستم در مورد محصولات پوکوپینی سوال بپرسم';
```

به:

```javascript
const message = 'سلام، میخوام سفارش بدم';
```

## مثال کامل


## تست آنلاین

میتوانید لینک را مستقیماً در مرورگر تست کنید:

```
https://wa.me/989121234567?text=سلام
```

اگر شماره صحیح باشد، واتساپ باز میشود.
