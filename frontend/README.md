# فرانت‌اند پوکوپینی

فروشگاه آنلاین لباس کودک - بخش فرانت‌اند

## تکنولوژی‌های استفاده شده

- React 19
- Tailwind CSS 3
- React Router DOM
- Axios
- Moment Jalaali

## نصب و راه‌اندازی

```bash
# نصب وابستگی‌ها
npm install

# اجرای سرور توسعه
npm start

# ساخت نسخه تولید
npm run build
```

## ویژگی‌ها

- طراحی واکنش‌گرا (Responsive)
- پشتیبانی کامل از زبان فارسی و RTL
- فونت فارسی Vazir
- تاریخ شمسی
- اعداد فارسی

## ساختار پروژه

```
src/
├── components/     # کامپوننت‌های قابل استفاده مجدد
├── pages/         # صفحات اصلی
├── contexts/      # Context API
├── services/      # سرویس‌های API
├── utils/         # توابع کمکی
├── App.js         # کامپوننت اصلی
└── index.js       # نقطه ورود
```
